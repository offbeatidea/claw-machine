# 测试用调试方法

> 本文档汇总抓娃娃机项目（v3.2.0）中所有用于测试和调试的方法。
> 适用文件：`index.html`, `code/js/main.js`, `code/js/claw.js`, `code/js/physics.js`, `code/js/dolls.js`

---

## 一、强制刷新检查

### 目的
确保浏览器加载的是**最新版本的 JS/CSS**，而不是缓存的旧文件。

### 方法
**每次修改 `index.html` 或 `code/` 下任何文件后，必须更新版本关键词！**

#### 步骤 1：更新版本关键词
在以下两个位置修改时间戳（格式：`v3.2.0-buildYYYYMMDD-HHMM`）：

**文件 1：`index.html` 第 16 行**
```javascript
console.log('%c[HTML] v3.2.0-build20260525-1701 - FORCE REFRESH CHECK', '...');
```

**文件 2：`code/js/main.js` 第 9 行**
```javascript
console.log('%c[Main] v3.2.0-build20260525-1701 - FORCE REFRESH CHECK', '...');
```

#### 步骤 2：浏览器强制刷新
| 浏览器 | 快捷键 |
|--------|----------|
| Chrome / Edge | `Ctrl + Shift + R` 或 `Ctrl + F5` |
| Firefox | `Ctrl + F5` |
| Safari | `Cmd + Shift + R` |

#### 步骤 3：验证
按 `F12` 打开控制台，确认看到：
```
[HTML] v3.2.0-build20260525-1701 - FORCE REFRESH CHECK
[Main] v3.2.0-build20260525-1701 - FORCE REFRESH CHECK
```

**如果看不到或版本号不对，说明缓存未清除，重复步骤 2。**

## 二、相机调试

### 1. 查看当前相机信息
在控制台（F12）输入：
```javascript
// 查看相机位置
camera.position

// 查看相机朝向（世界坐标系方向向量）
camera.getWorldDirection(new THREE.Vector3())

// 查看场景中物体数量
scene.children.length

// 查看场景中所有物体
scene.children
```

### 2. 调整相机位置与朝向
使用全局调试函数 `setCamera`：

```javascript
/**
 * 调整摄像机位置与朝向
 * @param {number} x - 相机X位置
 * @param {number} y - 相机Y位置
 * @param {number} z - 相机Z位置
 * @param {number} lookAtX - 看向点的X坐标（可选，默认0）
 * @param {number} lookAtY - 看向点的Y坐标（可选，默认4）
 * @param {number} lookAtZ - 看向点的Z坐标（可选，默认0）
 */

// 示例 1：默认视角
setCamera(5, 8, 12, 0, 4, 0)

// 示例 2：俯瞰视角（从正上方往下看）
setCamera(0, 20, 0, 0, 0, 0)

// 示例 3：侧面视角
setCamera(15, 5, 0, 0, 2, 0)

// 示例 4：正面视角
setCamera(0, 5, 15, 0, 2, 0)

// 示例 5：近距离特写
setCamera(3, 3, 3, 0, 2, 0)
```

### 3. 初始化时自动打印相机信息
游戏加载完成后，控制台会自动输出：
```
[Camera] 位置: Vector3 {x: 5, y: 8, z: 12}
[Camera] 朝向: Vector3 {x: ..., y: ..., z: ...}
[Camera] 场景物体数量: 12
[Debug] 调试用指令: setCamera(x, y, z, lookAtX, lookAtY, lookAtZ)
```

## 五、游戏循环调试

### 自动调试输出
游戏循环（`animate()` 函数）每 60 帧会自动输出一次日志：
```javascript
if (Math.floor(clock.elapsedTime * 60) % 60 === 0) {
    console.log('[Animate] 循环中... delta:', deltaTime.toFixed(3), 'children:', scene.children.length);
}
```

### 手动暂停/继续游戏循环
```javascript
// 暂停游戏循环（停止 requestAnimationFrame）
// 方法：在控制台输入以下代码覆盖 animate 函数
window._animateRunning = false;

// 继续执行（需要刷新页面）
location.reload();
```

## 七、配置面板调试

### 动态生成的配置项
配置面板会在初始化时动态生成以下配置项（每个都有解释说明）：

| ID | 标签 | 说明 |
|----|------|------|
| `grabSuccessRate` | 抓取成功率 | 爪子尝试抓取时，成功抓住娃娃的概率 |
| `strongGrabRate` | 强抓几率 | 抓取成功时，触发"强抓"的概率 |
| `dollSizeFactor` | 娃娃尺寸系数 | 控制所有娃娃的显示大小 |
| `clawSizeFactor` | 爪子尺寸系数 | 控制爪子的显示大小 |
| `clawInitialPos` | 爪子初始位置 | "center"为机箱中心，"exit"为出口上方 |
| `dropCheckCount` | 掉落判断次数 | 爪子移动过程中，每帧检测娃娃是否掉落的次数 |
| `dropChance` | 掉落率 | 每次掉落检测时，娃娃从爪子中掉落的概率 |
| `clawMass` | 爪子质量 | 影响爪子移动时的惯性 |
| `clawFriction` | 爪子摩擦力 | 爪子停止输入后，速度衰减的速度 |
| `clawForce` | 爪子推动力 | 按键时爪子的加速度 |
| `clawMaxSpeed` | 最爪子最大速度 | 爪子移动的速度上限 |
| `ropeLength` | 绳子长度 | 爪子绳子的长度 |
| `pendulumGravity` | 钟摆重力 | 爪子摆动时的重力强度 |
| `pendulumDamping` | 钟摆阻尼 | 爪子摆动的能量损耗 |

### 在控制台中查看/修改配置
```javascript

// 修改配置（示例：修改抓取成功率为 80%）

// 应用配置（如果需要立即生效，可能需要触发特定函数）
// 具体取决于配置项的实现方式
```

## 八、常见调试场景

### 场景 1：看不到 3D 物体（只有紫色背景）
**可能原因**：
1. 相机位置/朝向不对
2. Three.js 加载失败
3. 物体创建失败（JS 报错中断）

**排查步骤**：
1. 按 F12 打开控制台，查看是否有红色错误
2. 输入 `scene.children.length` 确认物体数量（应该 > 0）
3. 输入 `setCamera(0, 20, 0, 0, 0, 0)` 切换到俯瞰视角
4. 如果还是看不到，检查 Three.js 是否加载成功：
   ```javascript
   typeof THREE // 应该返回 "object"
   ```

### 场景 2：爪子不移动
**可能原因**：
1. 输入处理未生效
2. 爪子速度被限制为 0
3. 游戏状态不是 'idle'

**排查步骤**：
1. 检查游戏状态：`window.gameState`
2. 检查爪子速度：`window.clawVelocity`
3. 检查输入状态：`joystickInput` 或 `keys`
4. 查看控制台是否有 `[Claw]` 相关日志

### 场景 3：娃娃不掉落/掉落异常
**可能原因**：
1. 物理引擎未正确更新
2. 重力加速度设置错误
3. 地面碰撞检测失败

**排查步骤**：
1. 查看物理引擎状态：`window.PhysicsEngine`
2. 检查重力加速度：`window.PhysicsEngine.gravity`（应该是负值）
3. 查看娃娃物理对象：`window.PhysicsEngine.getDollPhysics(0)`
4. 查看控制台 `[PhysicsEngine]` 相关日志

### 场景 4：JS 修改后不生效
**原因**：浏览器缓存了旧 JS 文件

**解决**：
1. 更新版本关键词（见**第一节**）
2. 强制刷新（`Ctrl + Shift + R`）
3. 确认控制台显示新的版本关键词

## 九、物理引擎调试

### 查看娃娃物理对象
```javascript
// 获取指定娃娃的物理对象
const dollPhysics = window.PhysicsEngine.getDollPhysics(dollIndex);

// 查看物理对象属性
console.log(dollPhysics);
// 重要属性：
// - state: 'resting' | 'falling' | 'bouncing' | 'grabbed' | 'settling'
// - velocity: THREE.Vector3（速度向量）
// - position: THREE.Vector3（位置）
// - onGround: boolean（是否着地）
// - bounceCount: number（已反弹次数）
```

### 手动触发抛物线掉落
```javascript
// 获取娃娃物理对象
const dollPhysics = window.PhysicsEngine.getDollPhysics(0);

// 手动设置初速度并触发掉落
dollPhysics.velocity.set(1, 5, 0); // 设置初速度
window.PhysicsEngine.applyParabolicDrop(dollPhysics, dollPhysics.velocity);
```

### 修改物理参数
```javascript
// 修改重力加速度（默认 -9.8）
window.PhysicsEngine.gravity = -15;

// 修改反弹阻尼（默认 0.6，值越大反弹越高）
window.PhysicsEngine.bounceDamping = 0.8;

// 修改地面摩擦力（默认 0.9）
window.PhysicsEngine.friction = 0.8;

// 修改空气阻力（默认 0.02）
window.PhysicsEngine.airResistance = 0.05;
```
