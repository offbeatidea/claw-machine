## 第2阶段：抛物线掉落演算

### 🎯 目标
将当前的**线性掉落**（`animateDollDrop()` 中的 `doll.position.y -= dropSpeed`）改为**物理正确的抛物线轨迹**，模拟弱抓时娃娃被松开后的自然掉落。

### 📊 物理模型

**当前问题**：
```javascript
// 当前代码（第1646-1661行）
animateDollDrop(doll) {
    const groundY = 0.5;
    const dropSpeed = 0.05;  // 匀速！不符合物理
    const drop = () => {
        if (doll.position.y > groundY) {
            doll.position.y -= dropSpeed;  // ❌ 匀速掉落
            requestAnimationFrame(drop);
        }
    };
    drop();
}
```

**改进方案**：引入重力加速度和初速度
```javascript
// 目标效果
velocityY = initialVelocityY;  // 初始向上速度（爪子松开时的反弹）
velocityY -= GRAVITY * deltaTime;  // 重力加速
positionY += velocityY * deltaTime;
```

### 🔧 技术实现

#### 2.1 修改 `animateDollDrop()` 函数（第1646行）

**输入参数**：
- `doll`: 娃娃对象
- `initialVelX`, `initialVelZ`（可选）：爪子松开时给的水平初速度（模拟"甩出"效果）

**核心算法**：
```javascript
animateDollDrop(doll, initialVelX = 0, initialVelZ = 0) {
    const groundY = 0.5;
    const GRAVITY = 9.8;  // 重力加速度（可调）
    let velocityY = 2.0;   // 初始向上速度（爪子松开时的"弹出"效果）
    let velocityX = initialVelX;
    let velocityZ = initialVelZ;
    const massFactor = 1.0 / (doll.userData.mass || 1.0);  // 质量影响
    
    const drop = () => {
        // 重力影响
        velocityY -= GRAVITY * 0.016;  // 假设60fps，deltaTime≈0.016s
        
        // 位置更新
        doll.position.y += velocityY * 0.016;
        doll.position.x += velocityX * 0.016;
        doll.position.z += velocityZ * 0.016;
        
        // 空气阻力（可选）
        velocityX *= 0.99;
        velocityZ *= 0.99;
        
        // 落地检测
        if (doll.position.y <= groundY) {
            doll.position.y = groundY;
            // 触发第3阶段的"下滑"逻辑
            this.startDollSliding(doll, velocityX, velocityZ);
            return;
        }
        
        requestAnimationFrame(drop);
    };
    drop();
}
```

## 第3阶段：动态掉落+下滑

### 🎯 目标
1. **动态掉落**：娃娃掉落后不是立即静止，而是在地面/其他娃娃上**滑动一段距离**
2. **下滑**：模拟娃娃堆的颗粒物质力学——上方的娃娃可能向下滑动

### 📊 物理模型

#### 3.1 滑动物理（地面）

**摩擦力模型**：
```javascript
// 地面摩擦力
frictionForce = μ * m * g;  // μ=摩擦系数，m=质量，g=重力
deceleration = frictionForce / m = μ * g;
velocity *= (1 - μ * deltaTime);  // 速度衰减
```

**实现**：
```javascript
startDollSliding(doll, velX, velZ) {
    const groundFriction = 0.3;  // 地面摩擦系数（可调）
    const stopThreshold = 0.01;    // 停止阈值
    
    let velocityX = velX;
    let velocityZ = velZ;
    
    const slide = () => {
        // 摩擦力减速
        const speed = Math.sqrt(velocityX * velocityX + velocityZ * velocityZ);
        if (speed < stopThreshold) {
            console.log(`[slide] 娃娃"${doll.userData.name}"滑动停止`);
            return;
        }
        
        // 速度衰减（摩擦力）
        velocityX *= (1 - groundFriction * 0.016);
        velocityZ *= (1 - groundFriction * 0.016);
        
        // 位置更新
        doll.position.x += velocityX * 0.016;
        doll.position.z += velocityZ * 0.016;
        
        // 边界检测（防止滑出机台）
        doll.position.x = Math.max(-2.5, Math.min(2.5, doll.position.x));
        doll.position.z = Math.max(-2.5, Math.min(2.5, doll.position.z));
        
        requestAnimationFrame(slide);
    };
    slide();
}
```

```javascript
        }
    });
}
```

## 第4阶段：射线下抓

### 🎯 目标
将当前的**X-Z平面距离判定**（`checkCollision()`，第1188行）改为**射线检测（Raycasting）**，实现更精确的抓取判定。

### 📊 技术对比

| 方案 | 当前 | 目标 |
|------|------|------|
| 判定方式 | X-Z平面距离 | 从爪子向下发射射线 |
| 精度 | 粗糙（圆形碰撞体） | 精确（射线命中检测） |
| 视觉效果 | 只要距离够近就判定 | 必须"看到"娃娃才能抓 |

### 🔧 技术实现

#### 4.1 Three.js Raycaster 基础

```javascript

const raycaster = new THREE.Raycaster();
```

#### 4.2 修改 `checkCollision()` 函数（第1188行）

**方案A：纯射线检测**
```javascript
checkCollision() {
    if (this.collisionJudged) return;
    if (this.currentState !== GameState.DESCENDING) return;
    
    const raycaster = new THREE.Raycaster();
    const clawWorldPos = new THREE.Vector3();
    this.clawSwing.getWorldPosition(clawWorldPos);
    
    // 向下发射射线（从爪子到地面）
    raycaster.set(clawWorldPos, new THREE.Vector3(0, -1, 0));
    raycaster.far = 5.0;  // 射线长度限制
    
    // 收集所有娃娃的mesh
    const allMeshes = [];
    this.dolls.forEach(doll => {
        if (doll.userData.isGrabbed) return;
        doll.traverse(child => {
            if (child.isMesh) allMeshes.push(child);
        });
    });
    
    const intersects = raycaster.intersectObjects(allMeshes, false);
    
    if (intersects.length > 0) {
        // 找到命中的娃娃（可能需要向上遍历找到Group）
        const hitMesh = intersects[0].object;
        let hitDoll = hitMesh;
        while (hitDoll.parent && !hitDoll.userData.id) {
            hitDoll = hitDoll.parent;
        }
        
        console.log(`[raycast] 命中娃娃"${hitDoll.userData.name}"`);
        this.onCollisionDetected([hitDoll]);
    } else {
        console.log(`[raycast] 未命中任何娃娃`);
    }
    
    this.collisionJudged = true;
}
```

**方案B：射线+距离混合（推荐）**
```javascript
checkCollision() {
    // 第一步：距离粗筛（性能优化）
    const clawWorldPos = new THREE.Vector3();
    this.clawSwing.getWorldPosition(clawWorldPos);
    
    const candidates = this.dolls.filter(doll => {
        if (doll.userData.isGrabbed) return false;
        const distance = clawWorldPos.distanceTo(doll.position);
        return distance < CONFIG.CLAW_COLLISION_RADIUS + CONFIG.DOLL_SIZE;
    });
    
    if (candidates.length === 0) {
        this.collisionJudged = true;
        return;
    }
    
    // 第二步：射线精确判定
    const raycaster = new THREE.Raycaster();
    raycaster.set(clawWorldPos, new THREE.Vector3(0, -1, 0));
    raycaster.far = 5.0;
    
    const candidateMeshes = [];
    candidates.forEach(doll => {
        doll.traverse(child => {
            if (child.isMesh) candidateMeshes.push(child);
        });
    });
    
    const intersects = raycaster.intersectObjects(candidateMeshes, false);
    
    if (intersects.length > 0) {
        const hitDoll = this.findDollGroup(intersects[0].object);
        this.onCollisionDetected([hitDoll]);
    }
    
    this.collisionJudged = true;
}

// 辅助函数：从mesh找到所属的doll Group
findDollGroup(mesh) {
    let node = mesh;
    while (node.parent && !node.userData.id) {
        node = node.parent;
    }
    return node;
}
```

```javascript
        const intersects = raycaster.intersectObjects(allMeshes, false);
        intersects.forEach(hit => {
            const doll = this.findDollGroup(hit.object);
            hitDolls.add(doll);
        });
    });
    
    if (hitDolls.size > 0) {
        this.onCollisionDetected(Array.from(hitDolls));
    }
}
```
