## 一、需求概述

实现物理正确的抛物线掉落系统，替代当前的线性掉落。包含5个核心功能点：

1. 抓取失败后的娃娃跳动效果
2. 娃娃间碰撞检测（防重叠）
3. 弱抓触发的水平抛物线掉落
4. 移动中掉落的抛物线轨迹
5. 落地/落娃的二次反弹

## 二、详细需求规格

### 2.1 抓取失败后的娃娃跳动

**触发时机**: 爪子闭合抓取判定失败（未抓到）时

**物理模型**:
- 给被判定的娃娃一个初始速度
- XZ平面：随机方向（0-360度）
- Y方向：向上60度角的初速度
- 初速度大小：小值（建议 0.02-0.05 单位/帧）

**计算公式**:
```javascript
// 随机水平角度
const angle = Math.random() * Math.PI * 2;

// 60度向上 = Y轴分量占 sin(60°) ≈ 0.866
const speed = 0.03; // 初速度大小
const upAngle = 60 * Math.PI / 180; // 60度

dollVelocity.x = Math.cos(angle) * speed * Math.cos(upAngle);
dollVelocity.y = speed * Math.sin(upAngle); // 向上分量
dollVelocity.z = Math.sin(angle) * speed * Math.cos(upAngle);
```

**物理模拟**:
- 重力加速度：-9.8 单位/秒²（转换为每帧：`-9.8 * deltaTime * deltaTime`）
- 空气阻力：速度 *= 0.98（每帧衰减2%）
- 终止条件：速度小于阈值（如 0.001）或 Y <= 地面高度

**边界处理**:
- XZ平面边界：限制娃娃在机箱范围内
- 若超出边界：速度XZ分量反转并衰减（模拟碰撞墙壁）

### 2.2 娃娃间碰撞检测

**目标**: 防止娃娃之间空间重叠，模拟真实物理堆积

**检测频率**: 每帧检测（仅在娃娃掉落/移动时）

**碰撞判定**:
```javascript
// 两个娃娃的碰撞检测
function checkDollCollision(doll1, doll2) {
    const dx = doll2.position.x - doll1.position.x;
    const dy = doll2.position.y - doll1.position.y;
    const dz = doll2.position.z - doll1.position.z;
    const distance = Math.sqrt(dx*dx + dy*dy + dz*dz);
    
    const minDistance = DOLL_RADIUS * 2; // 两个娃娃半径之和
    
    if (distance < minDistance) {
        // 发生碰撞，计算排斥力
        const overlap = minDistance - distance;
        const nx = dx / distance; // 法线方向
        const ny = dy / distance;
        const nz = dz / distance;
        
        // 将娃娃推开
        const pushForce = overlap * 0.5;
        doll1.position.x -= nx * pushForce;
        doll1.position.y -= ny * pushForce;
        doll1.position.z -= nz * pushForce;
        doll2.position.x += nx * pushForce;
        doll2.position.y += ny * pushForce;
        doll2.position.z += nz * pushForce;
        
        // 速度交换（简化弹性碰撞）
        // ...
    }
}
```

**优化策略**:
- 空间分区：将机箱划分为网格（如 5x5x3），只检测相邻网格的娃娃
- 休眠机制：静止的娃娃（速度<阈值）跳过检测

**参数配置**:
| 参数 | 默认值 | 说明 |
|------|--------|------|
| DOLL_RADIUS | 0.3 | 娃娃碰撞半径 |
| COLLISION_DAMPING | 0.7 | 碰撞能量损失（0-1） |
| SLEEP_THRESHOLD | 0.001 | 休眠速度阈值 |

### 2.3 弱抓触发的水平抛物线掉落

**触发时机**: 爪子上升到顶部后，触发"弱抓"（强抓判定失败）

**物理模型**:
- 给娃娃一个随机水平方向的初速度
- 无垂直初速度（从爪子位置自由落体）
- 抛物线轨迹完全由重力和初速度决定

**实现步骤**:
```javascript
// 弱抓触发时
if (grabResult === 'weak') {
    const angle = Math.random() * Math.PI * 2; // 随机水平方向
    const horizontalSpeed = 0.02 + Math.random() * 0.03; // 0.02-0.05
    
    dollVelocity.x = Math.cos(angle) * horizontalSpeed;
    dollVelocity.y = 0; // 无初始垂直速度
    dollVelocity.z = Math.sin(angle) * horizontalSpeed;
    
    dollState = 'falling'; // 进入掉落状态
}
```

**掉落动画**:
- 娃娃从爪子位置（顶部）开始掉落
- 受重力影响：Y速度每帧增加 `-9.8 * deltaTime`
- XZ速度受空气阻力影响：每帧 `*= 0.99`
- 旋转动画：掉落时娃娃绕X轴或Z轴旋转（翻滚效果）

### 2.4 移动中掉落的抛物线轨迹

**触发时机**: 爪子移回出口过程中，娃娃掉落（掉落判定触发）

**物理模型**:
- 以**脱离点的速度**作为初速度
- 脱离点 = 爪子当前的位置和速度
- 抛物线轨迹由脱离速度 + 重力决定

**关键代码**:
```javascript
// 移动中掉落
function onDollDropDuringMove() {
    // 获取爪子当前速度
    const clawVelocity = {
        x: (clawTarget.x - clawBase.position.x) * moveSpeed,
        y: (clawTarget.y - clawBase.position.y) * moveSpeed,
        z: (clawTarget.z - clawBase.position.z) * moveSpeed
    };
    
    // 娃娃初速度 = 爪子速度 * 0.8（模拟"甩出"效果）
    dollVelocity.x = clawVelocity.x * 0.8;
    dollVelocity.y = clawVelocity.y * 0.8; // 可能向上或向下
    dollVelocity.z = clawVelocity.z * 0.8;
    
    dollState = 'falling';
}
```

**特殊情况**:
- 若爪子静止时掉落：使用小随机速度（同2.1）
- 若爪子快速移动时掉落：速度较大，抛物线轨迹更远

### 2.5 落地/落娃的二次反弹

**触发时机**: 娃娃掉落到地面或其他娃娃上时

**物理模型**:
- 弹性碰撞：速度Y分量反转并衰减
- 摩擦力：XZ速度衰减
- 多次反弹，直到能量耗尽

**实现代码**:
```javascript
function onDollLanding(doll) {
    // 检查是否落地或落娃
    const groundY = 0; // 地面高度
    const otherDollTop = checkOtherDollTop(doll); // 检测是否落在其他娃娃上
    
    const landingY = otherDollTop || groundY;
    
    if (doll.position.y <= landingY + DOLL_HEIGHT / 2) {
        // 触地
        doll.position.y = landingY + DOLL_HEIGHT / 2;
        
        // Y速度反转并衰减（弹性碰撞）
        dollVelocity.y = -dollVelocity.y * BOUNCE_DAMPING;
        
        // XZ速度衰减（摩擦力）
        dollVelocity.x *= FRICTION;
        dollVelocity.z *= FRICTION;
        
        // 若Y速度很小，停止反弹
        if (Math.abs(dollVelocity.y) < 0.01) {
            dollVelocity.y = 0;
            bounceCount++;
            
            if (bounceCount >= MAX_BOUNCES || 
                (Math.abs(dollVelocity.x) < 0.001 && Math.abs(dollVelocity.z) < 0.001)) {
                dollState = 'settled'; // 静止
            }
        }
    }
}
```

**参数配置**:
| 参数 | 默认值 | 说明 |
|------|--------|------|
| BOUNCE_DAMPING | 0.6 | 反弹能量保留（0.6 = 保留60%） |
| FRICTION | 0.9 | 地面摩擦力（每帧XZ速度*0.9） |
| MAX_BOUNCES | 3 | 最大反弹次数 |
| SETTLE_THRESHOLD | 0.01 | 静止判定阈值 |

**视觉优化**:
- 反弹时播放"挤压"动画（娃娃缩放Y轴）
- 落地粒子效果（可选）

## 三、技术实现方案

### 3.1 娃娃状态机

```javascript
const DOLL_STATES = {
    RESTING: 'resting',      // 静止在地面/其他娃娃上
    GRABBED: 'grabbed',      // 被爪子抓住
    FALLING: 'falling',      // 抛物线掉落中
    BOUNCING: 'bouncing',    // 反弹中
    SETTLED: 'settled'       // 已静止（可重新抓取）
};
```

每个娃娃需要存储:
- `position`: THREE.Vector3
- `velocity`: THREE.Vector3
- `state`: DOLL_STATES
- `bounceCount`: number
- `rotationSpeed`: THREE.Vector3（旋转速度）

---

### 3.2 物理更新函数

```javascript
function updateDollPhysics(doll, deltaTime) {
    if (doll.state === DOLL_STATES.RESTING || doll.state === DOLL_STATES.GRABBED) {
        return;
    }
    
    // 应用重力
    doll.velocity.y -= 9.8 * deltaTime;
    
    // 应用空气阻力
    doll.velocity.x *= (1 - AIR_RESISTANCE);
    doll.velocity.z *= (1 - AIR_RESISTANCE);
    
    // 更新位置
    doll.position.x += doll.velocity.x * deltaTime;
    doll.position.y += doll.velocity.y * deltaTime;
    doll.position.z += doll.velocity.z * deltaTime;
    
    // 更新旋转（翻滚效果）
    doll.rotation.x += doll.rotationSpeed.x * deltaTime;
    doll.rotation.z += doll.rotationSpeed.z * deltaTime;
    
    // 边界检测
    constrainToCabinet(doll);
    
    // 碰撞检测（娃娃间）
    checkAllDollCollisions(doll);
    
    // 落地检测
    checkLanding(doll);
}
```

## 六、风险提示

### 6.1 技术风险
- **性能问题**: 娃娃数量增多时，碰撞检测可能成为性能瓶颈
  - 缓解措施：空间分区 + 休眠机制
  
- **物理不稳定性**: 多个娃娃堆叠时可能出现抖动
  - 缓解措施：设置最小分离距离 + 速度阈值

### 6.2 设计风险
- **抛物线轨迹可能过于随机**: 玩家可能无法预测娃娃掉落位置
  - 缓解措施：提供"预测轨迹"可视化（可选）

## 附录：物理公式参考

### 抛物线运动
```
x(t) = x0 + vx * t
y(t) = y0 + vy * t - 0.5 * g * t²
z(t) = z0 + vz * t
```

### 弹性碰撞（Y方向）
```
vy_after = -vy_before * damping
```

### 摩擦力（XZ平面）
```
vx_after = vx_before * friction
vz_after = vz_before * friction
```
