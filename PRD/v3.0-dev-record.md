## v3.0 核心目标

1. **惯性系统**：爪子移动有质量感，不瞬移
2. **钟摆柔性悬挂**：爪子（clawSwing）通过绳子柔性悬挂在底座（clawBase）下方，移动时爪子因惯性滞后摆动（"甩抓"效果）
3. **相对位置绑定**：抓取成功时保持娃娃与爪子的相对位置（模拟抓到边角）
4. **模型标签**：Debug 用，显示模型名称和坐标

---

## 模型结构（最终版）

```
this.clawBase (Group) ← 玩家直接控制移动（键盘/触摸）
  ├─ top（底座模型，固定在 base 内）
  ├─ rope（绳子，动态拉伸连接底座与爪子本体）
  └─ this.clawSwing (Group) ← 做钟摆摆动
        ├─ fingers（爪指，3个）
        ├─ centerLine
        └─ clawGroundMarker（红色圆环，地面落点标记）
```

**关键设计**：
- 玩家控制 `clawBase.position` → 底座即时响应
- `clawSwing.position` = 钟摆偏移（滞后于底座）→ 实现"甩抓"效果
- 绳子每帧动态拉伸，视觉连接底座底部和 `clawSwing` 当前位置

## 已完成修改汇总

### 1. 惯性系统（2026-05-20）

**CONFIG 新增参数**：
```javascript
CLAW_MASS: 2.0,      // 爪子质量（越大越难加速）
CLAW_FRICTION: 0.85,  // 摩擦力系数（越小滑行越远）
CLAW_FORCE: 0.08,     // 推动力系数
CLAW_MAX_SPEED: 0.3    // 最大速度
```

**流程**：输入 → 施加力 → 更新速度 → 更新位置
**公式**：`velocity += (input * FORCE / MASS); velocity *= FRICTION; position += velocity`

---

### 2. 钟摆柔性悬挂系统（2026-05-21）

**核心物理**（在 `updateClawPosition()` 中每帧执行）：

```javascript
// 1. 计算底座加速度（驱动钟摆）
const accX = (baseX - prevBaseX) * ACCEL_FACTOR;
const accZ = (baseZ - prevBaseZ) * ACCEL_FACTOR;

// 2. 钟摆物理更新（速度反向受加速度影响）
this.pendulumVelX -= this.pendulumAngleX * GRAVITY * 0.016;
this.pendulumVelX -= accX;  // 关键：反向！底座左移 → 爪子右摆
this.pendulumVelX *= DAMPING;  // 阻尼
this.pendulumAngleX += this.pendulumVelX;

// 3. 计算爪子偏移
const offsetX = Math.sin(pendulumAngleX) * ROPE_LENGTH;
this.clawSwing.position.set(offsetX, offsetY, offsetZ);
```

**CONFIG 新增参数**：
```javascript
ROPE_LENGTH: 1.5,          // 绳子长度（钟摆绳长）
// currentConfig 新增：
pendulumGravity: 2.0,        // 钟摆重力
pendulumDamping: 0.97,       // 钟摆阻尼
```

### 3. 绳子动态拉伸（2026-05-21）

绳子从 `clawBase.add(rope)` 改为每帧动态拉伸：

```javascript
// updateClawPosition() 中
const ropeStart = new THREE.Vector3(0, baseBottomY, 0);  // 底座底部
const ropeEnd = new THREE.Vector3(offsetX, offsetY, offsetZ); // clawSwing 位置
const ropeMid = ropeStart.clone().add(ropeEnd).multiplyScalar(0.5);
const ropeDist = ropeStart.distanceTo(ropeEnd);

this.rope.position.copy(ropeMid);
this.rope.scale.y = ropeDist;  // 几何高度1.0，直接缩放
this.rope.quaternion.setFromUnitVectors(new THREE.Vector3(0,1,0), dir);  // 旋转对齐
```

### 4. 相对位置绑定（2026-05-20）

**抓取时**（`onCollisionDetected()`）：
```javascript
const offsetX = doll.position.x - clawWorldPos.x;
const offsetY = doll.position.y - clawWorldPos.y;
const offsetZ = doll.position.z - clawWorldPos.z;
this.grabbedDolls.push({ doll, offsetX, offsetY, offsetZ });
```

**移动时**（`updateClawPosition()`）：
```javascript
item.doll.position.set(
    clawWorldX + item.offsetX,
    clawWorldY + item.offsetY,
    clawWorldZ + item.offsetZ
);
```

### 5. 模型标签（2026-05-21）

**功能**：点击 Debug 面板中的"🏷️ 模型标签"按钮，显示/隐藏所有模型的名称和坐标。

**参数位置**（如要调整，直接改数字）：
- `createModelLabel()` 中：
  - `canvas.width = 256;` / `canvas.height = 128;` → 改分辨率
  - `sprite.scale.set(0.5, 0.25, 1);` → 改视觉大小
  - `material.opacity = 0.7;` → 改透明度
- `updateModelLabel()` 中：
  - `ctx.font = 'bold 28px Arial';` → 改标题字号
  - `ctx.font = '20px Arial';` → 改坐标字号
  - `canvas.height/2 - 20` → 改标题Y位置
  - `canvas.height/2 + 25` → 改坐标Y位置

### 6. Bug修复记录

| Bug | 根因 | 修复 |
|-----|------|------|
| `doll is not defined` | `moveToExit()` 变量名写错 | `doll` → `droppedDoll` |
| `Cannot read properties of undefined (reading 'set')` | `grabbedDolls[]` 存的是 `{doll,...}` 对象，但直接当 mesh 用 | `pop()` 后取 `item.doll` |
| 爪子不见了 | `clawSwing` 创建后没有 `clawBase.add(clawSwing)` | 补上 `add` |
| 绳子显示异常 | `ropeEnd` 的 Y 坐标错误加了 `baseBottomY` | `ropeEnd = (offsetX, offsetY, offsetZ)` |
| 移动关系反了 | 加速度对速度的影响方向错了 | `+= accX` → `-= accX` |
