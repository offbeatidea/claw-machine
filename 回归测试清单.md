## 三、回归测试方法

### P0 — 得分系统（F13, F14）
1. 启动游戏，确认初始得分 = 0，剩余次数 = 10
2. 抓取一个娃娃并成功放入出口
3. **预期**：得分立即变为 1，剩余次数变为 9
4. 继续抓取直到次数为 0
5. **预期**：次数为 0 后，得分不再变化，无法再抓取

### P0 — 强抓/弱抓判定（F12）
1. 打开配置面板，设置 `strongGrabRate = 1.0`（100% 强抓）
2. 抓取多个娃娃
3. **预期**：所有娃娃均为强抓，跟随爪子移动到出口
4. 设置 `strongGrabRate = 0`（0% 强抓）
5. 抓取多个娃娃
6. **预期**：所有娃娃均为弱抓，在移动过程中掉落

### P1 — 抓取动画（F9）
1. 抓取娃娃，观察爪子下降到底部后的动作
2. **预期**：爪指正确合拢（绕 X 轴旋转）
3. 上升后观察爪子移动到出口上方
4. **预期**：爪子正确张开

### P1 — 多娃娃跟随（F10）
1. 安排多个娃娃聚集在一起
2. 一次抓取多个娃娃
3. **预期**：所有娃娃同时跟随爪子移动，无掉落或分离

### P1 — 娃娃落地高度（F4）& 影子（F5）
1. 开始游戏，观察娃娃初始位置
2. **预期**：娃娃站在地面上（Y 坐标 = groundY + dollSize * 0.5）
3. 抓取娃娃并释放，观察落地后影子的位置
4. **预期**：影子在娃娃正下方，无偏移

### P2 — 配置实时生效（F17）
1. 打开配置面板
2. 拖动「绳子长度」滑条
3. **预期**：爪子绳子长度立即变化（无需关闭面板或刷新）

### P2 — 落地平移（F19）
1. 让娃娃自由落体到地面
2. **预期**：娃娃落地后 1~2 秒内完全静止，无持续平移
